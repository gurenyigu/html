什么是HTML5

HTML5是最新的HTML标准

HTML5是专门为承载丰富的WEB内容设计的，并且无需而外插件

HTML5拥有新的语义 图形 及 多媒体元素

HTML5提供的新元素的新的API简化了web应用程序的搭建

HTML5是跨平台的,被设计为在不同类型的硬件上运行

HTML5中默认的字符编码是UTF-8

HTML5新的属性语法

HTML5标准允许4种不同的属性语法

类型 示例
空属性 <input type="text" value="John Doe" disabled>
Unquoted <input type="text" value=John Doe>
双引号 <input type="text" value="John Doe">
单引号 <input type="text" value='John Doe'>

HTML5新特性:

    HTML5的一些最有趣的新特性
  • 新的语义元素, 比如: header footer article section 等元素
  • 新的表单控件,比如 number date color week range
  • 强大的图像支持(canvas svg)
  • 强大的多媒体支持(video audio)
  • 强大的新API,比如用本地储存取代cookie

HTML5 被删元素

    以下HTML4.0 元素已经在新的HTML5中被删除:
  • acronym
  • applet
  • basefont
  • big
  • center
  • dir
  • font
  • frame
  • frameset
  • noframes
  • strike
  • tt

HTML5浏览支持

所有现代浏览器都支持HTML5

鉴于浏览器会把未识别元素当作行内元素处理.所以可以如下操作:

        header,section,footer,aside,nav,main,aeticle,figure {
            display: block;
        }
