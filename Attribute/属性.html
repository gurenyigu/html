Attribute
属性 描述 支持
id

规定HTML元素的唯一 id

<element id="value">

id 在HTML文档中必须是惟一的

 All
class

指定元素一个或多个类

<element class="value">

class属性呢不能在这些元素中出现: base head html meta param script style title

类名不能以数字开头, 虽然IE支持

 All
title

规定元素的额外信息

<element title="text">

这些信息通常会在鼠标移动到元素上时显示一段工具提示文本

title属性常与 form a 元素一同使用. 也是 abbr acronym 元素的必须属性.

 All
lang

规定元素内容的语言

<element lang="language_code">

lang属性在以下标签中无效: base, br, frame, frameset, hr, iframe, param, script

 All
dir

规定元素中的文本显示方向

<element dir="ltr|rtl">

dir属性在以下标签中无效: base br frame hr frameset iframe param script

 All
style

规定行内样式(inline style)

<element style="key:value;key1:value1;">

其属性将覆盖全局的样式设定

 All
tabindex

指定元素 tab键的切换次序(当tab用于导航时

<element tabindex="unmber">

支持元素: a, area, button, input, object, select, textarea

 All
contenteditable

规定元素内容是否可编辑

<element contenteditable="true|false">

此元素可以从父元素继承

 All
data-*

用于存储页面或应用程序的私有定制数据。

<element data-*="value">

属性用于储存页面或应用程序的私有定义数据

  • 属性名不应该包含任何大写字母, 并且在前缀"data-*"之后必须有至少一个字符
  • 属性值可以是任意字符串
 用户代理会忽略前缀为 data- 的属性. All
accesskey

指定元素激活(默认行为)的快捷方式

<element accesskey="charracter">

支持元素: a, area, button, input, label, legend, textarea

使用Alt + accessKey (或者 Shift + Alt + accessKey) 来访问带有指定快捷键的元素。

accesskey="k"

 除了Opera
draggable

规定元素是否可拖动

<element draggable="auto|true|false">

链接和图像默认是可拖动的

 IE9+ 及其他
hidden

规定隐藏元素

<element hidden="hidden">

 IE11+ 及其他
spellcheck

规定是否对元素进行拼写和语法检查

<element spellcheck="true|false">

  • input元素中的文本值(非密码)
  • textarea 元素中的文本
  • 可编辑元素中的文本
 IE10+ 及其他
contextmenu

规定元素的上下文菜单, 在右键时出现上下文菜单

<element contextmenu="menu_id">

 仅支持 Firefox9+
dropzone

dropzone 属性规定在元素上拖动数据时, 是否拷贝, 移动或链接被拖动

<element dropzone="copy|move|link">

 没有支持
translate

规定元素内容是否应该被翻译

<element translate="yes|no">

使用 class="notranslata" 代替

 没有支持
