inherit

该值 可以使属性继承祖先设置.(IE7+)

inherit属于CSS-wide关键字，这表示所有的属性都可以接受该值

div {
  position: relative;
}

div a {
  position: inherit;
}

string

字符串(含转义字符串)

字符串可以由单/双引号包裹

转义字符由 \ 声明, 如: \"

URL

是对一个资源的指针, 在网页上提供一个资源地址.

由于历史原因, 可以对url不加引号: 

background-image: url(xx.png) | url("xx.png");

自定义取值 identifier

使用用户定义表示名作为组件取值

定义的标识 区分大小写, 不可以使用CSS中的占用词

/* 场景:  */
@keyframes onemove {
...
}

CSS3:

initial

恢复属性初始默认值(IE不支持)

initial属于CSS-wide关键字，这表示所有的属性都可以接受该值

ul {
  list-style: none;
}
.news-list {
  list-style: initial;
}
/* ul会被reset掉list-style，如果你想某个地方需要UA初始定义，那么就可以直接设置为initial  */

unset

擦除属性声明(支持不好)

一个属性定义了unset值，如果该属性是默认继承属性，该值等同于inherit，如果该属性是非继承属性，该值等同于initial unset属于CSS-wide关键字，这表示所有的属性都可以接受该值
