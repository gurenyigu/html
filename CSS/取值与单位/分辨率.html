resolution

单位: dpi, dpcm, dppx

  • 普通屏幕? 高分屏幕/视网膜屏? :

    普通屏幕通常包含96dpi，一般将2倍于此的屏幕称之为高分屏，即大于等于192dpi的屏幕，比如Mac视网膜屏就达到了192dpi（即2dppx），打印时一般会需要更大的dpi；

dpi

每英寸包含点的数量(dots per inch)

1dppx = 96dpi

1dpi ~= 0.39dpcm

1dpcm ~= 2.54dpi

1in = 2.54cm = 25.4mm = 101.6p = 72pt = 6pc = 96px

@media screen and (min-resolution: 192dpi) {
  .retina::befor {
    content: "视网膜屏幕";
  }
}	
@media screen and (min-resolution: 72dpi) and (max-resolution: 191dpi) {
  .normal::befor {
    content: "普通屏幕";
  }
}

dpcm

没厘米包含点的个数

dppx

每像素包含点的数量(dots per pixel)
