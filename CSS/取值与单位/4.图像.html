这不是一个CSS3新属性, 但是在CSS3中得到了增强.

<image> 取值可以为 url image() image-set() gradient

<image>值表示一个2D图像, 可以是url引用, 图像符号, 渐变符号

image 可以用在包括: border-image-source, background-image, list-style-image, cursor等属性上, 他被用来取代 url属性值.

image()

有些场景下, 图像可能是无效的, 比如当 url 指向的是一个无效的图像资源时. 一个无效的图像将表现为全透明, 但是当无效图像需要一些特殊行为时,可以使用image()

image() 用于指定图像或者图像替代

没有浏览器支持这个特性, 更多查看 
例如: 使用背景图片可以通过 background-position 来决定背景从哪个位置开始显示:
div {
  background-image: url(xx.png);
  background-position: 10px 10px;
}

现在亦可以这样做: 
div {
  background-image: image(xx.png#xywh=10, 10, 20, 20);
}
这样可以以图像(10, 10)为坐标原点, 截取一个20x20的图像显示

image-set

image-set() 可以根据用户设备的分辨率匹配合适的图像

支持不好 
div {
	background-image: image-set( "test.png" 1x, "test-2x.png" 2x, "test-print.png" 600dpi );
}

上述代码将会为普通屏幕使用 test.png，为高分屏使用 test-2x.png，如果更高的分辨率则使用 test-print.png，比如印刷

径向渐变参考文章

<gradient>

共有四种: linear-gradient() repeating-linear-gradient() radial-gradient() repeating-radial-gradient()

gradient 允许使用简单的语法实现颜色渐变，以便UA在渲染页面自动生成图像。

渐变在一个拥有尺寸的盒子中被生成，被称之为渐变盒，但是渐变本身并没有内在的尺寸，也就说如果在一个没有尺寸的容器上定义渐变，将无法被呈现。gradient 可以应用在所有接受图像的属性上

linear-gradient

下述值表示渐变方向,可以使用角度或者关键字来设置

  • 角度单位来指定方向
  • to left: 设置渐变: 从右到左, == 70deg
  • to right: 设置渐变从左到右, == 90deg
  • to top: 设置渐变从该上到下, == 0deg
  • to bottom: 设置渐变从上到下. == 180deg. 默认值

color: 指定起至颜色

length | percentage: 用百分比指定起止色位置.

用线性渐变创建图像.

如果想创建以对角线方式渐变的图像，可以使用 to top left 这样的多关键字方式来实现。

默认渐变: linear-gradient(#fff 10%, #333 40%, #233);
设置方向: linear-gradient(to left right, #666, #233, #888);
设置角度: linear-gradient(230deg, #456, #789 50%, #321);

每个颜色后不设置百分比起止色位置, 则默认50%

仅支持IE10+ 其他浏览器需要 hack IE6-9私有实现方案

radial-gradient()

取值:

position, 用来取定圆心位置. 如果提供2个参数，第一个表示横坐标，第二个表示纵坐标；如果只提供一个，第二值默认为50%，即center

  • 百分比: 指定径向渐变圆心的 横/纵 坐标. 可以为负值.
  • 用长度值: 指定径向渐变圆心的 横/纵 坐标. 可以为负值.
  • top: 指定顶部为径向渐变圆心的 纵 坐标
  • right: 指定右边为径向渐变圆心的 横 坐标
  • tottom: 设置底部为径向渐变圆心的 纵 坐标
  • left: 设置左边为径向渐变圆心的 横 坐标.
  • center: 设置中间为径向渐变的 横/纵 坐标

shape 圆类型

  • circle: 指定圆形的径向渐变
  • ellipse: 指定椭圆形的径向渐变

size: circle | ellipse均接受的值.

  • closest-side: 指定径向渐变的半径长度为从圆心到离圆心最远的边
  • closest-corner： 指定径向渐变的半径长度为从圆心到离圆心最近的角
  • farthest-side： 指定径向渐变的半径长度为从圆心到离圆心最远的边
  • farthest-corner： 指定径向渐变的半径长度为从圆心到离圆心最远的角

circle 接受该值作为 size

length：用长度值指定正圆径向渐变的半径长度。不允许负值。

ellipse 接受该值作为 size

  • length：用长度值指定椭圆径向渐变的横向或纵向半径长度。不允许负值。
  • percentage： 用百分比指定椭圆径向渐变的横向或纵向半径长度。不允许负值。

color-stop 用于指定渐变的起止颜色：

  • color： 指定颜色。
  • length： 用长度值指定起止色位置。不允许负值
  • percentage： 用百分比指定起止色位置。不允许负值
