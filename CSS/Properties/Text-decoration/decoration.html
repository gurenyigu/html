属性 版本 继承 描述
text-decoration css1/3 无 复合属性(不支持IE)。检索或设置对象中的文本的装饰
所有浏览器均支持CSS2.1中的 ' text-decoration ' 属性，在CSS3中，该属性定义被移植到其新的分解属性 ' text-decoration-line ' 上；

取值: 

                    text-decoration-line: 指定文本装饰的种类。
                    text-decoration-style: 指定文本装饰的样式
                    text-decoration-color: 指定文本装饰的颜色
text-decoration-line css3 无 检索或设置对象中的文本装饰线条的位置

取值: 

                none: 指定文字无装饰
                underline: 指定文字的装饰是下划线
                overline: 指定文字的装饰是上划线
                line-through: 指定文字的装饰是贯穿线
                blink: 指定文字装饰是闪烁
text-decoration-color css3 无 检索或设置对象中的文本装饰线条的颜色

取值:

取值为 color. 指定线条的装饰色
text-decoration-style css3 无 检索或设置对象中的文本装饰线条的形状

取值: 

                solid: 实线
                double: 双线
                dotted: 点状线条
                wavy: 波浪线
text-shadow css3(IE10+) 无 设置或检索对象中文本的文字是否拥有阴影及模糊效果

取值: 

                none: 无阴影
                length1: 第一个长度值用来设置对象的阴影水平偏移值。可以为负值
                length2: 第二个长度值用来设置对象的阴影垂直偏移值。可以为负值
                length3: 如果提供了第三个长度值则用来设置对象的阴影模糊值。不允许负值
                color: 设置对象的阴影颜色.
可以设置多组(三个长度+颜色)参数用间隔隔开，设置多组阴影效果
