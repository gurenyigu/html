属性 版本 描述
text-transform css1/3 检索或设置对象中的文本的大小写

取值: 

                none: 无转换
                capitalize: 将每个单词第一个字母转换成大写
                uppercase: 将每个单词转换成大写
                lowercase: 将每个单词转换成小写
                full-width: 将所有字符转换成fullwidth形式。如果字符没有相应fullwidth形式，将保留原样。这个值通常用于排版拉丁字符和数字订表意符号(css3)
white-space css1 设置或检索对象内空格的处理方式

取值: 

                normal: 默认处理方式
                pre: 用等宽字体显示预先格式化的文本，不合并文字间的空白距离，当文字超出边界时不换行。
                nowrap: 强制在同一行内显示所有文本，合并文本间多余空白，知道文本结束或者有br对象。
                pre-wrap: 用等宽字体显示预先格式化的文本，不合并文字间的空白距离，当文字碰到边界时发生换行.(IE8+)
                ptr-line: 保持文本的换行，不保留文字间的空白距离，当文字碰到边界时发生换行.(IE8+)
word-wrap css1 设置或检索当内容超过指定容器的边界时是否进行断行
css3更名为overflow-wrap。支持IE6+

取值: 

                    normal: 允许内容顶开或溢出指定的容器边界
                    break-word: 内容将在边界内换行。如果需要，单词内部允许断行.
overflow-wrap css3 设置或检索当前内容超过指定容器的边间时是否进行断行
word-break css3 设置或检索对象内文本的字内换行行为
这是IE5率先实现的属性。

取值: 

                normal: 依照亚洲语言和非亚洲语言的文本规则，允许在字内换行
                keep-all: 与所有非亚洲语言的normal相同。对于中文 韩文 日文 不允许字段开。适合包括少量亚洲文本的非亚洲文本。
                break-all: 该行为与亚洲语言的normal相同。也允许亚洲语言文本行的任意字内断开。该值适合包括一些非亚洲文本的亚洲文本，比如使连续的英文字符间断行。
text-align css1/3 设置或检索对象中内容的水平对齐方式

取值: 

                IE6+
                left: 内容在左对齐
                right: 内容右对齐
                center: 内容中间对齐
                CSS3: 
                justify: 内容两端对齐，但对于强制打断的行(被打断的这一行)及最后一行(包括仅有一行文本的情况，因为它既是第一行也是最后一行)不做处理
                

                start: 内容对齐开始边界
                end: 内容结束边界
                match-parent: 这个值和inherit表现一致，知识该值继承的start或end关键字是针对祖先 direction值计算的，计算可以是left和right
                justify-all: 效果等同于justify，但还会让最后一行也两端对齐.
                
大部分浏览器要使得 ' text-align ' 的justify两端对齐生效，需要在汉字间插入有空白，如空格；
块内的最后一行文本（包括块内仅有一行文本的情况，这时既是第一行也是最后一行）及被强制打断的行，其两端对齐需使用 ' text-align-last '；
IE浏览器下，如果 ' text-align-last ' 要生效，必须先定义 ' text-align ' 为justify；
word-spacing css1/3 检索或设置对象中的 单词 之间的最小，最大和最佳间隙

取值: 

                normal: 默认间隔
                length: 用长度值指定间隔。可以为负值.
                percentage: 用百分比指定间隔。可以为负值.(CSS3)
letter-spacing css1/3 检索或设置对象中的 字符 之间的最小或最大和最佳间隙

取值: 

                normal: 默认间隔
                length: 用长度值指定间隔，可以为负值
                percentage: 用百分比指定间隔。可以为负值(CSS3)
text-indent css1/3 检索或设置对象中的文本的缩进

取值: 

                length: 用长度指指定文本的缩进。可以为负值.
                percentage: 用百分比指定文本的缩进。可以为负值
                css3:
                each-line: 定义缩进作用在块级容器的第一行或者内部的每个强制换行的首行，软换行不受影响(CSS3)
vertical-align css1/2 唯一这里没有继承型的属性。设置或检索对象内容的垂直对齐方式

取值: 

                baseline: 将支持valign特性的对象的内容与基线对齐。
                sub: 垂直对齐文本的下标.
                super: 垂直对齐文本的上标。
                top: 将支持valign特性的对象的内容与对象顶端对齐.
                text-top: 将支持valign特性的对象的文本与对象顶端对齐
                middle: 将支持valign特性的对象的内容与对象中部对齐。
                bottom: 将支持valign特性的对象的文本与对象底端对齐。
                text-bottom: 将支持valign特性的对象的文本与对象顶端对齐.
                percentage: 用百分比指定由极限算起的偏移量。可以为负值。基线对于百分比数来说就是0%.
                length: 用长度值指定由基线算起的偏移量。可以为负值。基线对于数值来说为0.(CSS2)
line-height css1 检索或设置对象的行高。即字体最底端与字体内部顶端之间的距离

取值: 

                normal: 允许内容顶开或溢出指定的容器边界。
                length: 用长度值指定行高。不允许负值。
                percentage: 用百分比指定行高，气百分比取值基于字体的高度尺寸。不允许负值.
                number: 用乘积因子指定行高。不允许负值。
text-justify css3 设置或检索对象内调整文本使用的对齐方式
text-align-last css3 设置或检索一个块内的最后一行(包括块内仅有一行文本的情况，这时既是第一行也是最后一行)或者被强制大段的行的对齐方式

取值: 

                auto: 无特殊对齐方式
                left: 内容左对齐
                center: 内容居中对齐
                right: 内容右对齐
                justify: 内容两端对齐
                start: 内容对齐开始边界
                end: 内容对齐结束边界
tab-size css3 检索或设置对象中的制表符的长度

取值: 

                integer: 用整数值指定制表符的长度.
                length: 用长度值指定制表符的长度。
                以上均不允许负值
text-size-adjust css3 检索或设置移动端页面中对象文本的大小调整
