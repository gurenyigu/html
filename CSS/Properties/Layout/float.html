float

float CSS属性指定一个元素应沿其容器的左侧或右侧放置，允许文本的内联元素环绕它

该元素从网页正常流中移除，尽管其仍然保持部分的流动性(与绝对定位相反)

使用float 意味着使用块布局, 他在某些情况下修改 display值的计算值 (制定为bloack)

但是对 flex 和 inline-flex 无效

float 属性的值被指定为单一的关键字，值从下面的值列表中选择
值 描述
left 表明元素必须浮动在其所在t的块容器左侧的关键字
right 表明元素必须浮动在其所在的块容器右侧的关键字
none 表明元素不进行浮动的默认关键字

浮动元素的定位方式

脱离标准流的元素会根据值向左 或 右平移，一直平移知道碰到所处容器的边框，或者另一个浮动元素
