作为指定值，除了定位和浮动元素和根元素。 在这两种情况下，计算的值可以是除指定的值之外的关键字。

版本 属性值 描述
基本值 CSS1 none

关闭一个元素的显示(对布局没有影响);其所有后代元素都也会被关闭显示.完当渲染时,该元素如同不存在.

要想让元素在隐藏的同时占据其原有的位置, 使用visibility属性
inline 让元素生成一个或多个行内元素盒
block 该元素生成一个块元素
list-item 该元素生成一个容纳内容和单独的列表行内元素盒的块状盒
拓展值 CSS2.1 inline-block 该元素生成一个块状盒,该元素随着周围内容流动,如同他是一个单独的行内盒子(表现像是一个被替换的元素)
表格模型值 CSS2.1 inline-table 内联样式表格
table 类似<table>元素。它定义了一个块级盒子
table-caption 这个元素的作用类似<caption>
table-cell 这个元素的作用类似<td>
table-column 这个元素的作用类似<col>
table-column-group 这个元素的作用类似<colgroup>
