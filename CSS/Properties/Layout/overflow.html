overflow属性指定当前内容溢出其块级容器时，是否剪辑内容，显示滚动条或者显示溢出的内容
值 (IE6+) 描述
visible 默认值，内容不会被修剪，会呈现在元素框之外
hidden 内容会被修剪，并且其余内容不可见
scroll 内容会被修剪，浏览器会显示滚动条以便查看其余内容
auto 由浏览器定夺，如果内容被修剪，就会显示滚动条
规定从父元素继承overflow属性的值
  • 这段文本将溢出承载他的盒子。因为属性设置为 visible. 可见的
  • 这段文本的溢出部分将隐藏，不可见。属性设置为 hidden. 隐藏的
  • 这段文本溢出部分将隐藏，并显示滚动条。属性设置为 scroll.
  • 这段文本继承父元素的 overflow属性. inherit
