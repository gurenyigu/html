尺寸(css2 需要IE6+)
属性 版本 描述
width css1 检索或设置对象的宽度。除了非置换(display:inline-block;)内联元素，table-row,table-row-group之外的所有元素. 对于img 来说只指定了width, 那么高度会根据图片尺寸等比例缩放
min-width css2 检索或设置对象的最小宽度. 如果其值大雨 max-width属性值.那么max-width的值会自动替换为min-width的值
max-width css2 检索或设置对象的最大宽度
height css1 设置或检索对象的最小高度
min-height css2 设置或检索对象的最小高度
max-height css2 设置或检索对象的最大高度
