属性 版本 描述
border css1 复合属性，设置对象边框的特性
border-top|left|bottom|right css1 复合属性，设置对象的特性
设置单个边框的复合值
border[-top|left|bottom|right]-width css1 设置或检索边框宽度
如果border-style 设置为 none或hidden时，其值为0
border[-top|left|bottom|right]-style css1 设置或检索边框样式

取值:

  • none: 无轮廓。将忽略border-color,border-width计算值为0.除非轮廓为图像 border-image
  • hidden: 隐藏边框.IE7及胰腺癌尚不支持
  • dotted: 点状轮廓线.IE6下显示为dashed效果
  • dashed: 虚线轮廓
  • solid: 实线轮廓
  • double: 双线轮廓
  • groove: 3D凹槽轮廓
  • ridge: 3D凸槽轮廓
  • inset: 3D凹边轮廓
  • outset: 3D凸边轮廓
border[-top|left|bottom|right]-color css1 设置或检索边框颜色
当border-style设置为none或border-width等与0此属性将被忽略
border-radius css3(IE9+) 设置或检索对象的圆角边框
设置圆角边框。做多接收八个值。中间用 / 分割。定义水平半径和垂直半径。值可以为长度或百分比。不允许负值。
border-top-left-radius css3 设置或检索对象左上角的圆角边框
做多接受两个参数用空格分割。分别为水平半径和垂直半径。
border-top-right-radius css3 设置或检索对象右上角的圆角边框
border-bottom-right-radius css3 设置或检索对象右下角的圆角边框
border-bottom-left-radius css3 设置或检索对象左下角的圆角边框
box-shadow css3 设置或检索对象阴影
语法: box-shadow: none(无阴影) | 长度值1(设置对线的阴影水平偏移值，可以为负值) 长度值2(用来设置对象的阴影垂直偏移值，可以为负值) 长度值3(可选，如果提供设置对象的阴影模糊值。不允许负值) 长度值4(可选，提供了则设置对象的阴影外延值，可以为负值) [inset(提供啧设置阴影类型为内阴影。没有该值默认为外阴影.)]
border-image css3(IE11+) 设置或检索对象的边框样式使用图像来填充
border-image-source css3 设置或检索对象的边框是否用图像定义样式或图像来源路径
border-image-slice css3 设置或检索对象的边框背景图的分割方式
border-image-width css3 设置或检索对象的边框厚度
border-image-outset css3 设置或检索对象的边框背景图的扩展
border-image-repeat css3 设置或检索对象的边框背景图的平铺方式
