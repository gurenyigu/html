长度单位:

参考

相对单位: em, ex, ch(CSS3), rem, vw, vh, vmax, vmin, %

绝对长度: cm, mm, q, in, pt, pc, px

vw, vh(自适应)

vw, vh 直接对应当前可视窗口的width height..

设置: width: 100vw; height: 100vh;

计算: 假设当前浏览器宽 1000px, 高800px; 1vw = 1000/100 = 10px; 1vh = 800/100 = 8px;

就相当于: width: 1000px; height: 800px;

这个单位不会像 % 一样受父元素约束.

这里引入 vmax, xmin: 前者指当前宽 高的大者.后者指当前宽高的小者.

em, rem

两者均为相对大小单位. 区别在于:

em: 当前行内文本字体尺寸大小. 这样的话就牵扯到继承.

如: 当前行内字体为 20px(来自父元素), 那么 10em = 10 * 20px = 200px;

rem: 相对于 html 标签. r = root(根). 也就是说,只要用rem作为单位. 都会以 html 标签设置为基础值.

ex

ex单位被定义为: 当前字体的 x-height(小写x的高度) 或者一个 em 的一半;

ch

ch单位被定义为 0-width(0字符的宽度)
