!important

提升指定样式规则的应用优先级

兼容IE6+ 但是在IE6中有注意事项: 

		div {
			color: #f00 !important;
			color: #000;
		}
		// 这时在IE6中 !important  没有生效, 应用后者

@charset

在外部样式表文件内使用. 指定该样式表使用的字符编码

该规则后面的分号是必需的，如果省略了此分号，会生成错误信息。IE6+

	@charset "utf-8";

@import

指定绝对或相对地址导入的外部样式表 及目标媒体

该规则必须在样式表头部最先声明。并且其后的分号是必需的，如果省略了此分号，外部样式表将无法正确导入，并会生成错误信息。

IE使用@import无法引入超过35条的样式表。

	@import url("global.css");
	@import url(global.css);
	@import "global.css";
// 三种引入方式  均可

指定媒体查询: 

	@import url(example.css) screen and (min-width:800px);
	@import url(example.css) screen and (width:800px),(color);
	@import url(example.css) screen and (min-device-width:500px) and (max-device-width:1024px);

@media

指定样式表规则用于指定的媒体类型和查询条件

IE8及以下只能实现CSS2中的部分, 即只可以设置媒体类型

媒体查询可以被用在CSS中的@media和@import 规则上, 也可以被用在HTML和XML中.

	@media screen and (width: 800px) { ... }
	@import url(example.css) screen and (width: 800px);
	<link media="screen and (width: 800px)" rel="stylesheet" href="example.css" />
	<?xml-stylesheet media="screen and (width: 800px)" rel="stylesheet" href="example.css" ?>
	
		使用示例

@font-face

设置嵌入HTML文档的字体

向兼容所有浏览器,需同时使用TureTpe(.ttf)、Web Open Font Format(.woff)、Embedded Open Type(.eot)、SVG(.svg)四种字体格式。

/* 示例, 将兼容所欲浏览器字体 */
@font-face {
  font-family: 'diyfont';
  src: url('diyfont.eot'); /* IE9+ */
  src: url('diyfont.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
  url('diyfont.woff') format('woff'), /* chrome、firefox */
  url('diyfont.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
  url('diyfont.svg#fontname') format('svg'); /* iOS 4.1- */
}

@keyframes

指定动画名称和动画没帧效果

定义效果时可以使用关键词指定开始与结束: from to, 也可 *%

IE10+

@keyframes testanimations {
	from { transform: translate(0, 0); }
	20% { transform: translate(20px, 20px); }
	40% { transform: translate(40px, 0); }
	60% { transform: translate(60px, 20); }
	80% { transform: translate(80px, 0); }
	to { transform: translate(100px, 20px); }
}
