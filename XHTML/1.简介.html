XHTML 是以 XML 格式编写的HTML

XHTML 指的是可拓展超文本标记语言

XHTML 与 HTML 4.01几乎是相同的

XHTML 是更严格更纯净的HTML版本

XHTML 是以 XML 应用的方式定义的HTML

XHTML 是 2001年1月 发布的W3C推荐标准

XHTML 得到所有主流浏览器的支持

为什么要使用XHTML

因特网上的很多页面包含了 劣质 的HTML

XML是一种必须正确标记且格式良好的标记语言

文档结构

  • XHTML DOCTYPE 是强制性的
  • <html> 中的 XML namespace 属性是 强制性的
  • <html> <head> <title> <body>

元素语法

  • XHTML元素必须正确嵌套
    • 错误 正确
    <b><i>some_text<b><i> <b><i>some_text<i><b>
  • XHTML元素必须始终关闭
    • 错误 正确
    <br>
    <hr>
    <img src="" alt="">    		 <br />
    <hr />
    <img src="" alt="" />
  • XHTML元素必须小写
    • 错误 正确
    <BODY>
    <P>some_text<P>
    </BODY>    		 <body>
    <p>some_text<p>
    </body>
  • XHTML文档必须有一个根元素

属性语法

  • XHTML 属性必须是小写
    • 错误 正确
    <table WIDTH="100"> <table width="100" />
  • XHTML 属性值必须用引号包围
    • 错误 正确
    <input checked>
    <input readonly>
    <input disabled>
    <input disabled>
    <input selected>    		 <input checked="checked">
    <input readonly="readonly">
    <input disabled="disabled">
    <input disabled="disabled">
    <input selected="selected">
  • XHTML 属性最小化也是禁止的

    • <!DOCTYPE ...> 是强制性的

    XHTML 文档必须进行 XHTML 文档类型声明 (XHTML DOCTYPE declaration)

            <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

        <html xmlns="http://www.w3.org/1999/xhtml">

        <head>
        <title>Title of document</title>
        </head>

        <body>
        ......
        </body>

        </html>

    如何从HTML转换到XHTML

    1. 向每张页面的第一行添加XHTL <!DOCTYPE>
    2. 向每张页面的html元素添加xmlns属性
    3. 把所有元素名改为小写
    4. 关闭所有空元素
    5. 把所有属性名改为小写
    6. 为所有属性值加引号
