表单中的元素包括以下几种:

input元素

将在 HTML input 中详细介绍

select 元素(下拉列表)

由 select标签包裹(包含一个 name 属性定义下拉列表名)

内容由 option 组成 value属性指定当前列表值; selected 属性指定默认选项(默认第一个)

<textarea>元素

该元素用于多行输入字段(文本域)

<button> 元素

此元素定义可点击按钮

HTML 5 新增表单元素

  • <datalist>
  • <keygen>
  • <output>
默认的, 浏览器不会显示未知元素. 新元素不会破坏页面

<datalist>

此元素为 <input> 元素规定预定义选项列表

用户会在他们输入数据时看到预定义选项的下拉列表

datalist 选项中的标签使用 <option> 但标签 value 进行值的定义即可

input 元素的 list 属性必须引用 datalist 元素的 id 属性. 才可以使用
Safari 或 IE9- 不支持 datalist

<keygen> 标签

此功能已从WEB标准中删除. 虽然支持,但是避免使用.

keygen 标签规定用于表单的密钥对生成器字段

当提交表单时,私钥储存在本地, 公钥发送到服务器
