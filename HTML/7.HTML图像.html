图像标签() 和源属性 (Src)

在HTML中, 图像由 img 标签定义. img 是空标签, 他只包含属性, 并没有闭合标签. 若要在页面中显示图像, 要使用源属性(src). src 值 "source". 源属性的值的图像的 URL地址

文本属性(Alt)

alt属性用来位图片定义一串可替换的文本, 当图像替换文本属性的值是用户定义的

当浏览器无法加载图片时, 替换文本属性告诉用户失去的图片信息.

title

在放上图片时显示的信息

align 对齐属性(值与本行中文本的对齐方式)

值为: top bottom middle. bottom为默认属性

特殊值为 left 与 right. 其浮动到文档的左边和右边

<img src="xxx.png" align="middle" />

width and height

这两个属性可以直接为这张图像设置宽高

<img src="xxxx.jpg" width="100" height="100" />

使用 a标签 嵌套 img标签 达到图片链接的作用

定制带有可点击区域的图像映射. map and area

area 元素用于嵌套在 map元素中使用. area 元素可以定义图像映射中的区域.

像锚点一样. img中的 usemap 属性可以引用 map 中的 id 或 name属性(这取决于浏览器), 所以我们应该同时向 map 中添加 id 和 name 属性

area标签.

属性 值 描述
alt text 图片的替换文本
coords 坐标值 定义可点击区域的坐标
href URL 定义此区域的目标URL
nohref nohref 从图像映射排除某个区域
shape default rect circ poly 定义区域的形状
target _blank _top _parent _self 规定何处打开 href 属性指定的目标 URL

HTML 与 XHTML 差异: 在 HTML中, area 没有结束标签, 而在 XHTML中要正确关闭

coords定义 用法

  1. 图像左上角坐标为 0,0
  2. coords属性规定区域的坐标
  3. coords属性 与 shape 属性配合使用, 这样来规定尺寸, 形状, 位置

area 标签的 coords 属性定义了客户端图像映射中对鼠标命案的区域的坐标. 坐标的数字及其含义取决于 shape属性中决定的区域形状. 可以将客户端图像映射中的超链接区域定义为矩形 圆形 多边形等

形状 shape属性 coords属性 详细
圆形 circle x,y,r 这里的 x, 和 y 定义了圆心的位置, r是以像素为单位的原型半径
多变形 polygon x1, y1, x2, y2, x3, y3, ..... 每一对"x, y"坐标都定义了多边形的一个顶点.例如: 定义三角形至少需要三组坐标. 多边形会自动封闭,因此在列表结尾不需要重复第一个坐标闭合.
矩形 rectangle x1, y1, x2, y2 第一个坐标是矩形的一个角的顶点坐标, 另一对坐标是对角的顶点坐标. 定义矩形实际上是定义带有四个顶点的多边形的一种简化方式.

提示: 当两个 area 出现了重叠, 那么会优先采用 先出现的 area. 并会忽略超过图像边界范围之外的部分.

href 定义 用法

href 属性规定区域中连接的目标

值 描述
URL 链接的URL, 可能的值:
  • 绝对URL-指向另一个站点(比如: href="https://www.baidu.com/baidu.png")
  • 相对URL-指向站点内的某个文件(href="baidu.png")
  • 锚URL-指向页面中的锚(href="#href")

nohref 定义 用法

nohref属性会定该区有没有相关链接

target定义 用法

target属性 规定区域中连接目标

值 描述
_blank 在新窗口中打开链接
_self 默认值. 在相同框架中打开链接
_parent 在父框架中打开链接
_top 在整个窗口中打开链接
framename 在指定的框架中打开被链接文档(参考HTML框架)

提示: 这里指的框架是 iframe 这样的窗口. 在其中调试即可直到区别
