iframe

所有主流浏览器均支持内联框架

iframe元素可以创建包含另外一个框架的内框架

HTML 4.1 Strict DTD 和 XHTML 1.0 Strict DTD 中, 均不支持iframe元素

打开百度
iframe 中的属性
属性 值 描述
align left right top middle bottom 不赞成使用, 样式代替
frameborder 1 or 0 规定是否显示变框
name frame_name 定义iframe名称, 可用配合 target 使用
srcdoc HTML_code 所有主流浏览器都支持 srcdoc属性(Opera需要12+ Safari需要5+).除了Internet Explorer. 显示在框架中的内容.必须是有效的HTML语法. 若浏览器支持 srcdoc属性, 则覆盖 src 属性(若已经设置)中的内容. 如果不支持 srcdoc属性, 则相应地会显示在 src属性(若已经设置)中的内容
sandbox(H5) IE9- Opera12- 不支持此特性
值 描述
空字符串 应用以下所有限制
allow-same-origin 允许 iframe 内容被视为与包含文档有相同的来源
allow-top-navigation 允许iframe 内容从包含文档导航(加载)内容
allow-forms 允许表单提交
allow-script 允许脚本执行
如果被规定为空字符串(sandbox=""),sandbox属性将会启用一系列对行内框架中内容的额外限制. sandbox属性的值既可以是一个空字符串(应用所有限制), 也可以是空格分割的预定义值
height or width pixels 规定内联框架高 宽
marginheight or marginwidth pixels 定义 iframe内 顶部和底部的边距 / 左侧和右侧的边距
scrolling
auto 默认值 需要的情况下需要滚动条
yes 始终显示滚动条(即使不需要)
no 从不显示滚动条(即使需要)
规定是否在iframe中显示滚动条
longdesc URL(没有浏览器支持) 规定一个页面,该页面包含了有关iframe 的较长描述
seamless 仅支持Opera12+ Chrome Safari5+ 使iframe看上去向包含文档的一部分(Chrome无效果)
