HTML 链接

查看HTML锚点使用

超链接可以是字串 也可以是图片. 可以通过点击它来进行转跳到新文档或文档中的某个位置

HTML 链接属性

使用 target 属性, 可以定义被链接的文档在何处显示默认会在当前页面进行显示. 但是如果添加 target="_blank"那么就会在新页面显示

使用锚点. 添加 name="label". 可以用id 来代替 name 属性, 同样有锚点属性

注意!:

href 链接要将子文件夹的结尾以 "/" 结尾. 因为若不这样, 例如: href="http://www.w3school.com.cn/html. 这样就会向服务器产生两次HTTP请求. 这是因为: 服务器会添加 "\"到这个地址, 然后创建一个新的请求: href="http://www.w3school.com.cn/html/"

提示:

命名使用锚点最好常用于大型文档开始位置设置目录. 然后为每个章节设置锚点

即使锚点未被找到,也不会出现错误, 只会定位到链接文档顶部
